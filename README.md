# SyScreenshoter

Simple application to make screenshots. One `*.exe` without additional files.
![screenshot](screenshot.png)


## Requirements

 * Windows 7+ (x64)


## Using

 * run
 * select area to capture
 * fix captured image (use pen or text tool)
 * save or copy to clipboard
